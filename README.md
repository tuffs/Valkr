# VALKR

---

Designing a web framework that will be infinitely customizable, elegant to write,
and simple to deploy. This project aims to standardize the way that applications
are served on the web.

I have tired of the old frameworks which accomplish little for the imagination when
building applications on the web.

## INSTRUCTIONS

---

- To run VALKR simply run:
- `$ node ./valkr <command> <arguments>`

- For help run:
- `$ node ./valkr help`
